# map will create a new array with the results of calling a provided function on every element in this array.

# filter will create a new array with all elements that pass the test implemented by a provided function.

Since filter accepts an Observable , and not an array, we have to convert our array of JSON objects from data.json() to an Observable stream. This is done with flatMap .

# Flatmap: transform the items emitted by an Observable into Observables, then flatten the emissions from those into a single Observable

[1,2,3].map(x => [x, x * 10]) // [[1, 10], [2, 20], [3, 30]] [1,2,3].flatMap(x => [x, x * 10]) // [1, 10, 2, 20, 3, 30]] Operator to Use? - Instance Operators
